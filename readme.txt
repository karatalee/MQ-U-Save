Keterbatasan perangkat:
- Tidak dapat terkena air
- Perlu waktu untuk memanaskan sersor saat pertama kali dihidupkan (30 detik)
- Pengisian daya lama (5V 1A) (0-100% sekitar 60 menit) kapasitas battery 2500mah

Cara pemakaian:
1. Menekan saklar hingga lampu indikator menyala
2. Pada saat pertama kali diyalakan buzzer akan berbunyi, tunggu hingga buzzer berhenti 
   berbunyi (sensor membutuhkan waktu untuk memanas).
3. Letakan alat di area yang ingin diuji
4. Terdapat tombol mute di sebelah kiri perangkat.(berfungsi untuk mematikan buzzer selama
   5 detik jika sensor terus menerus mendeteksi adanya gas)
5. Segera lakukan charging jika battery habis (indikator battery terdapat pada layar)
6. Lampu led akan berwarna merah jika dalam proses charging, dan akan berubah menjadi hijau jika 
   battery penuh.(matikan perangkat saat proses charging!)
